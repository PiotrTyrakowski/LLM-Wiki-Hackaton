# Pacing — baseline rules

You are deciding how fast the edit should feel. Pacing is shot-length distribution over time.

## Rules

1. **Open fast.** First 5-10 seconds should average shot length ≤ 2.5 seconds — hook the viewer with motion and change.
2. **Slow down for the point.** When the speaker makes the key argument, extend the held shot to 6-10 seconds. Stillness signals importance.
3. **Vary, don't lock in.** Never let three shots in a row have the same duration. Variance is rhythm.
4. **Cut faster during energy peaks.** Music drops, applause, demos, action — drop shot length below 2 seconds.
5. **End on a held shot.** The last 3 seconds before the title card should hold without a cut.

## Output contract

When asked to evaluate pacing, return per-segment shot length stats and a one-line verdict.
